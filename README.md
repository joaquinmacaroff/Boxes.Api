Boxes Api
============
[![.NET 10](https://img.shields.io/badge/.NET%2010-blueviolet.svg)](https://dotnet.microsoft.com/download/dotnet/10.0)

Boxes is an RESTfull API written in .NET 10 to manage customer appointments for workshops.

### Main Technology Stack

- **.NET 10**: Latest LTS framework
- **FastEndpoints**: REPR pattern for API endpoints

### Setting your own appsetting 
Please use as guide the default `appsetting.json` for create yor own `appsetting.Environment.json`.

### Build and Run

```powershell
# Build the solution
dotnet build

# Run the application
dotnet run --project src
```

### Swagger 
With the project running, go to `https://localhost:{your-port}/swagger/index.html` for read the autogenerated swagger documentation for available endpoints.

If you want in `./docs` you can restore a postman collection
